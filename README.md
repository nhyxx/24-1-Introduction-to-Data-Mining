# 스니커즈 리셀 시장 판매자들을 위한 투자 상품/전략 추천
서울과학기술대학교 산업정보시스템전공 데이터마이닝 팀 프로젝트

### 팀원
- 산업정보시스템전공 이혜린
- 산업정보시스템전공 박현길
- 기계시스템디자인공학 오민석
- 건설시스템공학 이태경

### 사용 언어
Python 

### 1. 배경 및 필요성 
MZ 세대 소비 성향이 변화하며 새로운 리셀 시장이 등장했고, 리셀 상품 판매는 "리셀테크"라고 불리며 새로운 재테크로 떠오르는 중

다른 재테크 방법들보다는 진입이 쉬우나 급격히 성장하게 된 시장으로 투자 상품과 관련한 정보가 부족하다는 단점

### 2. 분석 목적
리셀 시장에 발을 들이고 싶어 하는 소비자, 리셀 시장에서 리셀테크를 진행 중인 투자자들을 위한 투자 상품 및 투자 전략 정보 제공을 목적으로 함

### 3. 사용 데이터 및 수집 방법
Kream 사이트 내 상품 정보들을 크롤링하였음
````
kream_data.csv
````
